# Funktionalität des Dashboards (Nando)

Das bereits bestehende Grafana Dashboard des TESS-W Photometer Networks soll nicht ersetzt oder verbessert werden. Es wird ein lokales Programm gefordert, welches spezifischere Analysen ermöglicht und die passenden Grafiken darstellt. Ausserdem soll der Export eines Reports durchgeführt werden können.

## Minimal Valuable Product (MVP) (Nando)

Gemäss den Forderungen wurde als Lösung ein Dashboard als Webapp definiert, welches bessere Analysetools und optimale Grafiken enthält. Folgend wird das minimale wertvolle Produkt beschrieben.

<li>Lokales Analyseprogramm, läuft auf jedem PC</li>
<li>Einbezug von Photometer Daten des TESS-Photometer Netzwerks</li>
<li>Darstellung von Grafiken einzelner Sensoren</li>
<li>Diverse Grafiken -> möglichst guter Überblick</li>
<li>Export Funktion als PDF-Bericht</li>
<li>Installation TESS-W Photometer auf FHNW-Dach -> Daten in Programm einbeziehen</li>
